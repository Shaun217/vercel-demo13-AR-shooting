# Cute Shape Sorter (No Camera Version)

一个可爱的黏土风格 3D 几何分类游戏，专为 Web 设计。

## 特点
- **可爱风格**: 马卡龙配色，哑光材质，圆角 UI。
- **无需摄像头**: 使用鼠标（电脑）或触摸（手机）直接拖拽积木。
- **纯前端**: 无需后端，直接部署。

## 如何运行
1. **GitHub + Vercel (推荐)**: 
   - 将这 4 个文件上传到 GitHub。
   - 在 Vercel 中导入该仓库。
   - 点击部署，即可获得在线链接。

2. **本地运行**:
   - 如果使用 VS Code，右键 `index.html` 选择 "Open with Live Server"。
   - 或者直接双击 `index.html` (因为移除了摄像头依赖，本地双击也可以运行！但推荐使用服务器以防贴图加载问题)。

## 玩法
1. 输入玩家人数。
2. 输入名字开始。
3. 用鼠标按住上方掉落的积木。
4. 拖拽到下方对应的白色盒子里 (三角形对三角形，正方形对正方形...)。
5. 速度越快，排名越高！
